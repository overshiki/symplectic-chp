# Bell State Derivation

## Circuit
```
H 0
CNOT 0 1
M 0
M 1
```

## Mathematical Derivation

### Step 1: Initial State
$$|\psi_0\rangle = |00\rangle$$

### Step 2: Hadamard on Qubit 0
The Hadamard gate creates a superposition:
$$H|0\rangle = \frac{|0\rangle + |1\rangle}{\sqrt{2}} = |+\rangle$$

So the state becomes:
$$|\psi_1\rangle = |+\rangle \otimes |0\rangle = \frac{|00\rangle + |10\rangle}{\sqrt{2}}$$

### Step 3: CNOT(0, 1)
The CNOT gate flips qubit 1 when qubit 0 is |1⟩:
$$\text{CNOT}(|00\rangle) = |00\rangle$$
$$\text{CNOT}(|10\rangle) = |11\rangle$$

Therefore:
$$|\psi_2\rangle = \frac{|00\rangle + |11\rangle}{\sqrt{2}} = |\Phi^+\rangle$$

This is the Bell state (maximally entangled).

### Step 4: Measurement
The Bell state $|\Phi^+\rangle$ is a +1 eigenstate of both $Z \otimes Z$ and $X \otimes X$:
- $(Z \otimes Z)|\Phi^+\rangle = +|\Phi^+\rangle$
- $(X \otimes X)|\Phi^+\rangle = +|\Phi^+\rangle$

When measuring both qubits in the Z-basis:
- The outcomes are perfectly correlated (both 0 or both 1)
- Each outcome has probability 1/2
- This is because $|00\rangle$ and $|11\rangle$ have equal amplitude

## Stabilizer Analysis

The Bell state has stabilizer group generated by:
- $g_1 = X \otimes X$  (destabilizer)
- $g_2 = Z \otimes Z$  (stabilizer)

In CHP tableau representation:
- Stabilizers: $\{+ZZ, +IX\}$ (rows 0, 1)
- Destabilizers: $\{+XX, +ZI\}$ (rows 0, 1)

Note: The exact phases depend on measurement outcomes due to randomness in CHP.

## Expected Properties

1. **Measurement Correlation**: M0 and M1 are always equal
2. **Stabilizers**: After preparation, the state satisfies $Z_0 Z_1 = +1$ and $X_0 X_1 = +1$
3. **Tableau Validity**: The symplectic basis maintains $\omega(D_i, S_j) = \delta_{ij}$
